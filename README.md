# 🔍 Dev Detective — GitHub User Search

**Week 3 Project | Prodesk IT Internship**
**Level: Intermediate (Level 2)**

Dev Detective is a simple web app that lets you search for any GitHub user and view their profile details along with their latest repositories.

## 📌 About

This project was built to get hands-on experience with APIs in a real-world scenario.

Instead of just fetching data and logging it in the console, I focused on building a complete flow — taking user input, calling the GitHub API, and updating the UI dynamically with the fetched data.

It helped me understand how frontend applications actually interact with external data sources.

---

## 🚀 Features

### 🔹 User Search
- Search any GitHub user using their username
- Displays:
  - Avatar
  - Name
  - Bio
  - Followers / Following
  - Public repositories count
  - Join date
- Direct link to open the profile on GitHub

---

### 🔹 Repositories
- Shows the latest 5 updated repositories
- Each repo includes:
  - Name (clickable link)
  - Star count
  - Language
  - Last updated date

---

### 🔹 UI & Experience
- Loading state while fetching data
- Handles errors (like invalid username) without breaking the app
- Smooth UI updates without page reload
- Clean dark-themed interface
